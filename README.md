# Character counter

## Overview
This project is a Character Counter application where the user can enter any text — a sentence, a word, or even a single character.
The app displays:
Character count
Word count
Estimated reading time
It’s designed to be fast, simple, and user-friendly.

## Features
* Count total characters
* Count total words
* Real-time updating as the user types
* Displays estimated reading time based on text length
